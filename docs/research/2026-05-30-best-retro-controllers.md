## Two ASIN gotchas the writer must know

1. **Pro 2 listing choice.** The first search hit for "8BitDo Pro 2" (`B0BFGSNLTM`, "Transparent Purple," sold by third-party "RunSnail," 4.6/1,899) is a reseller relisting. The pick below uses `B0CSPH1JYV` (Gray Edition) instead because its H1 is the clean current title, it ships from and is sold by Amazon.com, and it is the standard colorway. Both are the same physical Hall-Effect-revision Pro 2; the Gray listing is the more canonical and stable one for an affiliate link.
2. **A Pro 3 now exists** (`B0FFGQ6J35`, ~$59, TMR sticks, swappable ABXY, charging dock). The task locks the Pro 2 as best overall and that holds on price and review volume, but the writer should add a one-line "there's a newer Pro 3 if you want the latest sticks and a dock" so the guide does not read as dated.
