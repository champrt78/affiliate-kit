## How to read this

- **Winnability** is a competition read, not a volume score: **HIGH** = page 1 is forums / Reddit / small blogs / brand blogs (a new authority site can break in). **MED** = mixed (some big editorial + lots of small sites). **LOW** = page 1 owned by huge backlinked sites (OutdoorLife, Field & Stream, TechGearLab, Live Science) with no soft spots. For review/vs pages, winnability is rated on the **long-tail modifier**, not the head "[product] review" term (Amazon + the brand always own that head, and that's expected — it doesn't make the page LOW).
- **The no-hunters filter is a real discriminator.** Most trail-cam volume is deer/scouting/hunting. The under-served, winnable lane for this site is **home security / property monitoring / backyard wildlife / bird watching**. Every recommendation below steers to that lane. Hunter-coded phrases are flagged and redirected to the homeowner variant.
- Every secondary phrase traces to an autocomplete suggestion actually observed. Every winnability call cites the SERP I read. Where I did NOT read the SERP, it's marked *(unverified — autocomplete only)*.
