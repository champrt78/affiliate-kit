# SEO Keyword Map — starteraquarium.com

**Site:** starteraquarium.com (brand-new domain, days old)
**Niche:** beginner freshwater aquarium gear (starter kits, filters, heaters, lighting) + fun finds + beginner how-to (nitrogen cycle)
**Date:** 2026-05-30
**Method:** Google autocomplete (suggestqueries API) for keyword discovery + live WebSearch SERP reads for winnability (no paid volume tool). Winnability is a SERP-difficulty judgment: a SERP owned by Aquarium Co-Op, PetMD/PetSmart/Petco, Wirecutter/Spruce, or YouTube = HARD. A SERP made of forums (Fishlore, AquariumAdvice, bettafish.com, Reef2Reef, PlantedTank), Reddit, Quora, and small/mid niche blogs = WINNABLE for a new site.

**Strategy note (anti-cannibalization):** The 6 existing buyers-guides stay as **category hubs** targeting the moderate-tail head ("best beginner aquarium filter"). The highest-value **tank-size + species** long-tails ("best filter for a 5 gallon betta") become **dedicated new pages**, not hub retrofits, so the hub and the new page never compete for the same query. Lower-volume size/species variants live as **H2/FAQ blocks inside the hub**. Each page below states which long-tails it owns vs. which it hands off.
