## Method & honesty notes

- **Demand signal (no paid volume tool):** Google autocomplete (via `suggestqueries.google.com` JSON endpoint), harvested 2026-05-30. A keyword is "real demand" if it (a) appears as an autocomplete suggestion, (b) spawns multiple `vs`/`for`/`under` variants, and (c) shows competitor pages built specifically to answer it. **No invented monthly-volume numbers** — that would be dishonest without a tool.
- **Winnability (replaces a difficulty score):** every HIGH/MED/LOW call below cites a SERP actually run via WebSearch on 2026-05-30. Rule applied: page-1 = Reddit / r/SBCGaming / niche retro blogs / comparison microsites / forums / YouTube → **winnable**; page-1 = IGN / RetroRGB / Wirecutter / GameSpot / TechRadar / Tom's Guide / marketplaces (Amazon, Etsy, Walmart, Best Buy) → **hard**.
- **The gold (confirmed):** handheld-vs-handheld and "best handheld for [system/budget]" SERPs are dominated by tiny comparison microsites (Retro Catalog, comparehandhelds, handheldsarena, thecoreplay, whirlchain, Joey's Retro Handhelds, RetroHandheldHQ) + Reddit + YouTube. **Zero mega-publishers on those pages.** This is where a focused new site genuinely competes. The retro-handheld long-tail is the heartland of this domain.
