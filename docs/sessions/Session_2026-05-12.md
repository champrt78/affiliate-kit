## Bootstrap CLI Bugs Found & Patched (committed `636692f`)
1. **`copy-template.ts` copied `node_modules/` from the template** — failed on Windows because pnpm workspace symlinks need Developer Mode / elevation to recreate. Added a filter to skip `node_modules`, `.astro`, and `dist`. pnpm install at the destination recreates dependencies cleanly.
2. **`cloudflare-r2.ts` blew up the whole bootstrap** when R2 wasn't enabled at the account level (error 10042). R2 is a Phase 2 image-hosting feature; the bootstrap should not require it. Now logs a warning and continues.
3. **`cloudflare-pages.ts attachDomain` called `wrangler pages domain add`** which was removed in wrangler 4.x. Replaced with a direct POST to `/accounts/{id}/pages/projects/{name}/domains`.
