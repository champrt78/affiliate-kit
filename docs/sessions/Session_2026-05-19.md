## Discoveries / Gotchas (added)

- **Wikimedia `Special:FilePath/<filename>?width=N`** is a stable image API that 302-redirects to the correct thumbnail on upload.wikimedia.org. Browsers follow it natively in `<img src>`. Don't try to construct the hash-based upload URL by hand — use Special:FilePath and let Wikipedia do the lookup. Useful any time we need to embed a verified public-domain image without scraping.
- The Edit tool `new_string` parameter is positional-strict — a typo (`new_style`) fails the whole call with InputValidationError; nothing partially applies. Reads as Claude-side handler issue, not a file-state issue.
