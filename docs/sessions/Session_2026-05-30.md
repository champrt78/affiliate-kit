### Discoveries / Gotchas
- **`fix-product-images.ps1` replaces by URL string globally** (`$content.Replace('"oldUrl"', '"newUrl"')`). Safe when URLs are unique/non-empty, but it CANNOT populate empty `image: ""` fields — an empty oldUrl matches every `""` in the file. The handhelds guide needed a per-ASIN block-local regex sub instead. If we want the fixer to handle empty images, rewrite its core to `[regex]::Replace` with a MatchEvaluator (block-local). TODO logged.
- **Heredoc gotcha bit again:** ran `git commit -m @'...'@` (PowerShell here-string) inside the **bash** tool — bash took `@'` literally, so the commit subject became a bare `@`. Amended with `git commit --amend -F -` + a real bash heredoc. Rule: in the Bash tool use `<<'EOF'`, never `@'...'@`.
