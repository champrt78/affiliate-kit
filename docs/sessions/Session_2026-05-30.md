### Discoveries / Gotchas
- **Mini-audit beats the fixer for tall products.** `fix-product-images.ps1` always takes `colorImages.initial[0]` (the main shot), which for heaters/bottles is a sliver that fails the 0.35..2.5 aspect gate. `.tmp/pick-square-imgs.ps1` decodes EVERY `hiRes`+`large` candidate and picks min |log(aspect)| within the gate. This is the Canopy-free equivalent of `audit-product-images.ps1`. Worth promoting into the pipeline for any tall-product category (heaters, bottles, controllers-on-end).
- **Python regex in a bash heredoc needs the quotes pre-normalized.** firecrawl rawHtml escapes quotes as `\"`; matching `\\"hiRes\\"` with `[^\\]` char-classes kept throwing "unterminated character set". Fix: `raw.replace('\\"','"')` first, then a plain `"(?:hiRes|large)":"(...)"` pattern. Also: `python3` is the MS Store stub on this box; use `python`. And `firecrawl` is a shell shim, not a real exe — can't `subprocess.run(["firecrawl",...])`; scrape in bash, parse the file in python.
