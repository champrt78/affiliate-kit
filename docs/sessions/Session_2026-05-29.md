- **SLOT-SCOPING GOTCHA (advisor catch, load-bearing):** Astro scopes a component's `<style>` to the markup IT authors. SiteShell only POSITIONS the brand slot. Everything else (nav/footer/panel) is authored+styled in SiteShell so its scoped styles are fine. DTP's MainLayout keeps the full `.brand*` block for exactly this reason.
- **`--page-scale` tokenized:** was an undefined per-site magic number (`var(--page-scale, .95)` with no definition anywhere). Now `:root{--page-scale:.95}` in `shared-styles/tokens.css`, documented as the per-site scale-override knob (a site overrides it in its own site-tokens.css). APPROVED 0.95 floating-panel zoom + mobile zoom:1 reset preserved exactly — NOT switched to a rem model (out of scope).
