## Discoveries / Gotchas

- **Avasreview's trust contradiction is structural, not editing.** "Hans, 25 years hunting experience, tested for weeks" alongside disclosed methodology of "meta-review" (collecting external tests + customer reviews + specs). The two sentences contradict directly. Any reader who reads both notices. We never claim hands-on, so we don't carry this liability — but it's a worth-knowing model of "what NOT to imitate" if we ever feel pressure to inflate trust signals.
- **Popular Mechanics URL slug `a37679766` is a Hearst CMS content-ID artifact.** Their domain authority (DA ~90+) compensates for the weak URL semantics — but it means we can't out-SEO them on broad "best trail cameras" with clean URLs alone. Route around via long-tail queries ("trail camera for backyard wildlife", "no-glow trail camera") where their generic framing is structurally disadvantaged.
- **PM (Hearst), Wirecutter (NYT) all block direct fetches** with WAF rules even for archive.ph and cache routes. The Round 2 agents pivoted to secondary sources (case studies, design analyses, brand documentation) and labeled inferred claims. Their reports still produced concrete actionable patterns — "research value: high" — but the analyst noted the limitation transparently. Worth knowing: when targeting NYT/Hearst properties via ce-web-researcher in the future, expect WAF blocks and instruct the agent to fall back to secondary analysis from start.
- **AllAboutBirds runs Mercury (H&Co commercial license).** Closest open-source analog for our register is Lora or Merriweather for body. Fraunces is decorative at display sizes — perfect for H1/H2 — but loses authority at small body sizes. The current Astro body font setting is worth re-checking against the AAB recipe (18px / 1.65 / 68-72ch) before any polish pass.
- **The Strategist won SPD Silver for web typography partly on the rubric eyebrow pattern alone.** One small-caps line above every H1. The single cheapest visible "publication" cue available. Trivial to add to our Astro templates as a frontmatter field.
- **Wirecutter's "Flaws but not deal-breakers" is the literal phrase, not a paraphrase.** It's a recognized craft move — the section header itself does the trust work, not the prose inside it. Adopting the exact phrasing matters.
- **`ce-web-researcher` returns are large (~1500 words each).** 10 of them is ~15k words of context. Synthesis required active triage to extract the highest-leverage cross-site patterns. Worth knowing for future multi-agent dispatches: smaller per-agent word budgets (under 1000 words each) might be more efficient at 5+ agent scale.
- **Playground skill's "dark theme" UI + light-themed preview area splits cleanly.** Controls panel (dark) on the left; preview pane (the actual cream + forest + brass mywildlifecam aesthetic) on the right; prompt bar (dark, monospace) at the bottom. Pattern worth re-using.
- **CSS toggle-switch click target trap (Ray caught it).** First playground build had the visible `.track` + `.knob` spans positioned absolutely AFTER the `<input>` in DOM order — same stacking context, no z-index. Result: the visual layers absorbed pointer events and the underlying checkbox never fired. Label-text click worked (via `for=""`), clicking the visible pill did not. Fix: `pointer-events:none` on `.track` and `.knob` + `z-index:2` on the input. Cheap pattern, easy to miss on a custom toggle. Worth knowing for any future skill/playground build.

## Discoveries / Gotchas (late marathon)

- **Spypoint maintains multiple Demandware catalog hashes** and not all paths resolve. `Sites-physical_product_m_catalog/default/dw5517eccf/...` works; `Sites-spypoint-master-catalog/default/dwc83d2c01/...` returned 404. Standardize on the working path when adding new Spypoint imagery.
- **`prefers-color-scheme: dark` auto-flipped the whole token palette** to charcoal on Windows dark-mode systems. Removed the auto-flip; site stays in its warm-paper register regardless of OS theme. `color-scheme: light` on `<html>` opts out of browser dark-mode form-input styling too.
- **Product image scale disparity** between Spypoint (manufacturer PNG, ~25% canvas) and Trailcampro composite (~70% canvas) made cards look wildly inconsistent. `mix-blend-mode: multiply` blends the hard-white retailer-composite backgrounds into the cream card; per-image `imageScale` (CSS transform) lets us bump the small-on-canvas products to match. Spypoint uses 1.8.
- **Astro auto-fit grids stretch unbounded** when item count is small. Capping `repeat(auto-fit, minmax(320px, 420px)) + justify-content: center` was the fix for review grids with 2-3 items; forced `repeat(3, 1fr)` with a 720px collapse-to-mobile breakpoint for the home-page Latest-reviews row.
- **The `playground:playground` skill's prompt-bar is unnecessary for same-session decisions** — Ray pointed this out after picking the Magazine 1+2 variation. Saved a feedback memory; future playgrounds skip the copy-prompt panel unless explicit hand-off to a future Claude session is the goal.
- **Affiliate Lab.zip (Matt Diggity's course) was a 25.7 GB truncated download** — central directory missing. Re-download required. Worth extracting the keyword-research SOP + backlinking process docs once recovered; skip the videos.
- **Windows + Python 3.14 + last30days = cp1252 encoding crash on emoji output.** Workaround: `PYTHONUTF8=1` env var. Worth knowing for any other Python skills/scripts that emit Unicode on Windows.
- **`yt-dlp` via `pip --user` doesn't land on PATH** until shell restart; winget install also requires shell restart for the new PATH entry. Workaround: invoke the binary via its full path inside `~/AppData/Local/Microsoft/WinGet/Packages/yt-dlp.yt-dlp_.../yt-dlp.exe` until a shell relaunches.
- **`Write` hook blocks files with `innerHTML` usage** — XSS-pattern security warning, even for trusted/static playground HTML. Workaround: use static DOM with `data-` attribute switching + CSS class toggle instead of JS innerHTML reassignment. Tighter playground architecture anyway.
- **`AIOS dashboard's frontend defaults to port 3000`** (Vite's default), not 5173 as the README says. Worth correcting the README + the desktop shortcut already points to 3000.
- **AIOS `python dashboard/server.py` exits immediately** — frontend showed "Could not reach the API" error. Correct invocation is `uvicorn server:app --port 8000` from inside `dashboard/`. Captured in the `start-aios.bat` launcher.

1. **Cards orphaned in a center column on both index grids.** `/reviews/` and `/buyers-guides/` used `repeat(auto-fit, minmax(300px, 380px)) + justify-content:center` — at 1280px that orphaned 2 cards (reviews) or 1 card (guides) into the middle with 600+px of dead horizontal space.
